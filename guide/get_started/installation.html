Installation

Mathematics.NET is available to download from nuget.

  • Package Reference
  • Polyglot Notebooks

To use Mathematics.NET in your project, add the following line to your .csproj file:

<PackageReference Include="Physics.NET.Mathematics" Version="" />

Please have Visual Studio Code installed as well as the Polyglot Notebooks extension installed to begin using Mathematics.NET. When that is complete, create a new .ipynb file and add the following line to import the package:

#r "nuget: Physics.NET.Mathematics"

To get a specific version of Mathematics.NET, use, for example,

#r "nuget: Physics.NET.Mathematics, *-*"
